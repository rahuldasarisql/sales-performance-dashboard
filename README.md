# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Overview

This project presents an interactive Sales Performance Dashboard built using Power BI. It helps analyze key business metrics such as total sales, profit, orders, and profit margin to support data-driven decision making.

---

## 🚀 Features

* KPI Cards: Total Sales, Total Profit, Total Orders, Profit Margin
* Sales Trend Over Time (Line Chart)
* Regional Sales Comparison (Bar Chart)
* Sales Distribution by Category (Pie Chart)
* Interactive Date Slicer for dynamic filtering

---

## 🛠 Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)

---

## 📈 Key Insights

* Top-performing region identified based on sales
* Electronics category contributes the highest share
* Sales trends vary across different time periods

---

## 🧩 Interactivity

This dashboard includes a date slicer that allows users to filter data dynamically. All visuals update automatically based on the selected date range.
